   ```bash
   git clone git@github.com:embat-consultora/onboardings.git
   ```
2. **Crear un entorno virtual de Python:**
   ```bash
   python -m venv venv
   ```
3. **Activar el entorno virtual:**
   ```bash
   venv\Scripts\activate
   ```
4. **Instalar las dependencias:**
   ```bash
   pip install -r requirements.txt
   ```
